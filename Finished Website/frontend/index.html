Tool Description

This tool allows you to encrypt or decrypt any text or number input using block cipher algorithm, ensuring your privacy and security.

How do I use this tool?

  1. Select Cipher.
  2. Enter the Plain Text or Ciphertext you would like to encrypt or decrypt.
  3. Select the format of the text (Plain Text or Hex).
  4. Enter the secret key to be used during the process.
  5. Select the output format (Plain Text or Hex).
  6. Click the "Encrypt" button if you enter the plain text or click the "Decrypt" button if you enter the cipher text.
  7. View the output result in the large text area on the right-hand side of the screen.
